# a2rps

Apache Requests Per Second - A Python script to analyze Apache log files and calculate requests per second.

## Description

This tool helps you analyze Apache web server log files to extract requests per second (RPS) metrics. It parses Apache access logs and provides statistics about traffic patterns over time.

## Requirements

- Python 3.x
- Apache access log files in standard formats (Common Log Format or Combined Log Format)

## Usage

```bash
python a2rps.py <path-to-apache-log-file>
```

## Features

- Parse Apache access logs
- Calculate requests per second
- Display traffic statistics
